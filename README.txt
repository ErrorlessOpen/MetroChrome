MetroChrome UWP Browser Project

This package contains the MetroChrome source project for Windows 10 Mobile.

To build the final APPX:
1. Open in Visual Studio 2019/2022
2. Install UWP tools + Windows 10 SDK
3. Build for ARM (phones) or x86 (emulator)
4. Use:
   Project -> Store -> Create App Packages

Target devices:
- Lumia 950
- Lumia 950 XL
- Lumia 650
- Lumia 640
- Lumia 640 XL
- W10M Emulator
